SpringMvc跳转方式

发表于 2019-01-18 | 分类于 SpringMvc

默认方式

SpringMvc默认将请求进行进行转发

添加redirect：资源路径 重定向（浏览器地址会发生改变）

@RequestMapping("demo9")
public String demo9() {
    System.out.println("重定项");
    return "redirect:main.jsp";
}

添加forward：资源路径 或省略forward：转发（浏览器地址不会改变）

@RequestMapping("demo10")
public String demo10() {
    return "forward:demo11";
}

@RequestMapping("demo11")
public String demo11() {
    System.out.println("demo11");
    return "main.jsp";
}
