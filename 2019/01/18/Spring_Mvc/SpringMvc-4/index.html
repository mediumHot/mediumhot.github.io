SpringMvc自定义视图解析器

发表于 2019-01-18 | 分类于 SpringMvc

自定义视图解析器

1 springmvc.xml添加下面的配置
效果：在返回值前会加上/在返回值后面会加上.jsp

<bean id="viewResolver" class="org.springframework.web.servlet.view.InternalResourceViewResolver">
    <property name="prefix" value="/"></property>
    <property name="suffix" value=".jsp"></property>
</bean>

2 如果希望不执行自定义视图解析器，在方法返回值前添加forward或redirect

@RequestMapping("demo10")
public String demo10() {
     //不加forward:跳转的是/demo11.jsp
    return "forward:demo11";
}    
@RequestMapping("demo11")
public String demo11() {
    System.out.println("demo11");
     //注意这里的return的写法
    return "main";
}
