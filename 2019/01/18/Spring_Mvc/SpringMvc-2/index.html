SpringMvc参数传递方式

发表于 2019-01-18 | 分类于 SpringMvc

一 SpringMvc参数传递

1. 基本数据类型参数

1 默认参数名称和请求的参数名称相同

 @Controller
public class DemoController {        
    @RequestMapping("demo")
    public String demo(String name,int age) {
        System.out.println("执行了demo "+name+" "+age);
        return "/main.jsp";
    }
}

2 如果请求参数名和方法参数名不对应可以使用使用@requestParam()
@RequestParam(value=”name1”)中的value的name1对应浏览器的参数名称

 @RequestMapping("demo")
public String demo(@RequestParam(value="name1")String name,@RequestParam(value="age1")int age) {
    System.out.println("执行了demo "+name+" "+age);
    return "/main.jsp";
}

3 如果方法参数是基本数据类型(不是封装类)可以通过@requestParam设置默认值
防止没有参数500错误（封装类没有参数会用null代替）

RequestMapping("page")
public String page(@RequestParam(defaultValue="2")int pageSize,@RequestParam(defaultValue="1")int pageNumber) {
    System.out.println(pageSize+" xxx "+pageNumber);
    return "main.jsp";
}

没有加参数

加部分参数

4 如果强制要求有参数

RequestMapping("page")
public String page(@RequestParam(defaultValue="2")int pageSize,@RequestParam(defaultValue="1")int pageNumber) {
    System.out.println(pageSize+" xxx "+pageNumber);
    return "main.jsp";
}


5 required和defaultValue一起使用，不传递参数情况下使用默认值

@RequestMapping("demo3")
public String demo3(@RequestParam(required=true,defaultValue="11")String name) {
    System.out.println(name);
    return "main.jsp";
}

2. HandlerMethod中参数是对象类型

1 类信息

public class Peolpe {
    private String name;
    private int age;
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public int getAge() {
        return age;
    }
    public void setAge(int age) {
        this.age = age;
    }
    @Override
    public String toString() {
        return "Peolpe [name=" + name + ", age=" + age + "]";
    }
}

2 JSP页面–请求参数名和对象中属性名对应(get/set方法)

<form action="demo4" method="post">
    <input type="text" name="name">
    <input type="text" name="age">
    <input type="submit" value="提交">
</form>

3 Controller

@RequestMapping("demo4")
public String demo4(Peolpe peolpe) {
    System.out.println(peolpe);
    return "main.jsp";
}

6. Restful传值方式（重要：开发中常用的一种方式）

优点：简化jsp中参数编写格式
1 在JSP中特地的格式

<a href="demo8/ab/123">跳转</a>

2 Controller
2.1 在@RequestMapping中一定要和请求格式对应
2.2 {名称}中名称自定义
2.3 @PathVariable获取@RequestMapping中内容，默认按照方法参数名称去寻找

    @RequestMapping("demo8/{name}/{id1}")
    public String demo8(@PathVariable (value="id1")int id,@PathVariable String name) {
        System.out.println(id+" "+name);
        return "/main.jsp";
    }
